## 快速入门

<!-- 在此次添加使用文档 -->

如需进一步了解，参见 [midway 文档][midway]。

### 本地开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 文件生成插件使用（oss-generate-crud）

```bash
$ npm run ogc
```

### 部署

```bash
$ npm start
$ npm stop
```

### 内置指令

-   使用 `npm run lint` 来做代码风格检查。
-   使用 `npm test` 来执行单元测试。
-   使用 `npm run ogc` 来执行生成 crud 接口。

[midway]: https://midwayjs.org
